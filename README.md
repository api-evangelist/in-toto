# in-toto (in-toto)
in-toto is a CNCF graduated framework for securing software supply chain integrity. It provides specifications for generating and verifying metadata about each step in the supply chain, ensuring authorized execution and tamper detection.

### in-toto Attestation Specification
The in-toto specification defines the metadata format for recording software supply chain steps including layout metadata and link metadata for verification.
